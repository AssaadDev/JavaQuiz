# JavaQuiz

Simple Java Console Quiz

This project is a basic Java console quiz application. It displays five multiple-choice questions to the user, collects their answers, and calculates the final score.
The program uses three classes:

Main – starts the quiz.

QuestionsService – stores questions, handles user input, and calculates the score.

Question – a model class that represents a single quiz question with its options and correct answer.

The quiz runs in the terminal, allowing the user to type in their answers and see their final result at the end.
